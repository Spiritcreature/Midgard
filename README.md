# Midgard


/*** Objectif du PROJET 5 ***/

Créer un site pour un Bar à Thème médiéval.

Le site devra avoir :
 
- Un menu fixe en haut du site avec le logo du bar sur la	gauche.
- Une page d'accueil qui présente le Bar, un texte de bienvenue, ainsi qu'un diaporama de diverses photos du bar (attention les photos ne seront pas contractuelles, le bar est encore en construction).
- Une page représentant la carte des boissons (idem carte non définitive mettre des exemples).
- Une page récupérant les commentaires laissés sur la page facebook du bar (page facebook non encore mise en lecture pour le publique).
- Une page expliquant que le bar peut organiser des soirées à thèmes. Cette page devra avoir un formulaire de réservation et devra utiliser datepicker.
- Et enfin une dernière page contenant les informations pratiques du bar. Son adresse, ses coordonnées, ainsi qu'une carte géolocalisant le bar.

	
Le client souhaite également avoir une interface admin pour pouvoir :

- Rajouter une boisson sur sa carte.
- Retirer/ Remettre la ou les boissons sélectionnée(s) sur la carte.
- Il veut voir aussi les réservations que ses clients ont faites et pouvoir les modifier ou cas où une des informations sont à changer / préciser. Il doit y avoir également la possibilité de les supprimer.
- Il aimerait également une fonction « pense bête » pour éventuellement avoir des alertes / messages de rappels qui apparaitront au moment où il se loguera. 

ATTENTION : Le client ne souhaite pas avoir de site fait avec wordpress. Il ne veut pas avoir un mettre à jour des tas de plugins ou encore se perdre dans les menus. Il veut interface simple et rapide.
